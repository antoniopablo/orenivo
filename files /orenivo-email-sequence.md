**Cuerpo:**

Hey!

You've been using Orenivo for a few days now. Here are 3 patterns I've seen power users adopt:

**1. Organize by project, not by platform**

Instead of a "ChatGPT" folder and a "Claude" folder, try:
- 📁 Project Alpha (chats from ChatGPT + Claude mixed)
- 📁 Client Work
- 📁 Personal/Learning

This way, all context for one project lives together — regardless of which AI you used.

**2. Pin your "reference" conversations**

That chat where you nailed the perfect system prompt? Pin it. The one where Claude explained that tricky concept perfectly? Pin it.

Pins = your personal AI knowledge base.

**3. Use the Prompt Manager (Pro feature)**

If you find yourself typing the same instructions over and over ("Act as a senior developer, review this code..."), save it as a template.

Pro tip: use variables like {language} or {topic} so one template works for many situations.
